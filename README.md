# 🎮 Tic Tac Toe Game

An interactive and responsive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.
This project helped me strengthen my understanding of **DOM manipulation, event handling, game logic, and responsive UI design** while building a fun and user-friendly web application.

---

## 🚀 Features

* ✅ Two-player gameplay
* ✅ Win and draw detection
* ✅ Reset game functionality
* ✅ Clean and responsive UI
* ✅ Smooth user interaction
* ✅ Built with pure vanilla JavaScript

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**

---

## 📚 What I Learned

While building this project, I explored:

* DOM manipulation
* Event listeners
* Conditional logic
* Array-based win conditions
* UI state updates
* Problem-solving through real project building

This project gave me better confidence in creating **interactive frontend applications**.
